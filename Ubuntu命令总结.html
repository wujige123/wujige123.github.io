开启ssh服务端： 更新软件列表 sudo apt-get update 安装ssh服务 apt-get install openssh-server 检查是否启动 ps -e | grep sshd 手动开启ssh服务 /etc/init.d/ssh start 手动关闭ssh服务 /etc/init.d/ssh stop 手动重启ssh服务 /etc/init.d/ssh restart 安装软件命令： 从软件源安装软件 sudo apt-get update sudo apt-get install 本地安装 dpkg -i xxx.deb 卸载软件命令： 查看已安装软件包 dpkg -l 从软件源来的卸载 sudo apt-get remove 本地卸载 dpkg -r --purge xxx.deb
